2. D-1 Case Retrieval Process

Assigned Staff retrieves all measles IPD cases reported on previous day (D-1 date) from both Google Sheets.

Example: Cases reported on 10/05/2026 are included in report prepared on 11/05/2026.
3
3. Case Compilation Process
  • Cases copied from both sheets
  • Cases appended into Master Line List
  • Master file stored in: "Measles Report 2026" Google Drive Folder
Example: Cases reported on 10/05/2026 are included in report prepared on 11/05/2026.
4
4. Data Cleaning & Standardization

Assigned Staff performs:

Duplicate checking
Reporting date entry
Missing data verification
Hospital name verification
Standard formatting correction
Address correction
5
5. Data Verification Process

Verified by Assigned Staff/Data Analyst:

Correct ward mapping
Admission status verification
Proper age classification
Completeness of records
Vaccination status accuracy
Consistency of reporting date
Duplicate removal confirmation
6
6. Final Line List Preparation
  • Cleaned and verified line list involved
  • Saved in "Measles Report 2026" folder
  • Daily line list prepared by 04:00 PM
7
7. Data Analysis & Reporting

Data Analyst prepares reports whenever requested by senior officials. Standard analyses includes:

Ward-wise cases
Vaccination status analysis
Highest burden wards
Public vs Private facility analysis
Age group-wise cases
Admission severity analysis
Gender-wise cases
ICU/Ventilator/Critical cases
IPD vs OPD analysis
Discharge and death status
